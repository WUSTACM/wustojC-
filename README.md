# WUSTOJ 大一C语言作业
📘 本项目收录了 武汉科技大学 (WUSTOJ) C语言大一课程的编程作业答案，采用 Markdown 文档形式进行整理与说明，方便同学们学习、参考与复盘。

## 🌐 在线访问
项目已部署在个人博客：[everlastingblog.top](https://everlastingblog.top/)  
你可以直接在网页上浏览所有题目与答案，无需下载或配置环境。

## 📂 项目内容
每个作业题目均有对应的 题目描述 与 参考答案

答案以 C语言源码 + Markdown说明 的形式呈现

部分题目附带 运行截图 或 测试样例，帮助理解代码逻辑

按照课程进度与 WUSTOJ 平台题号进行分类整理

## 🚀 使用方式
在线浏览  
直接访问 everlastingblog.top 查看题目与答案。

本地查看

克隆仓库到本地：

```bash
git clone https://github.com/你的用户名/你的仓库名.git
```

使用 Markdown 阅读器或 VS Code 打开 .md 文件即可。

## 🎯 项目目标
帮助大一同学快速理解 C语言基础知识与编程思路

提供清晰的题解与代码示例，避免重复踩坑

作为学习笔记与知识库，方便后续复习与扩展

## ⚠️ 注意事项
本项目仅供学习交流使用，请勿直接抄袭提交

建议在参考答案的基础上，结合自己的理解进行修改与优化

如果发现错误或有更优解，欢迎提交 Issue 或 Pull Request
